Barebones Demo Page for Wellio.js

Uploads local LAS files using a file loader, turns them into json using wellio.js, and displays logs using g3.js.

First: use the 'browse' button to select LAS files on your computer.

Choose File...  

Second: click 'load' button to load LAS file(s) into browser memory.

Third: convert .las text into JSON. If you selected more than one las, the first will be used.

no well selection

Display Results as original text (A), converted JSON (B), or visual curves (C)
